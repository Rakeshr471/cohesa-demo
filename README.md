# Cohesa Demo

This is a minimal Vite + React + Tailwind project containing your app.
It includes the extra libraries used by the component: framer-motion, lucide-react, recharts.

## Run locally
```bash
npm i
npm run dev
```

## Build
```bash
npm run build
```
